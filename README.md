# nana-utils-bot

## 1. Telegram Bot Token Verifier (with History & Auth)

A Flask-based tool to:
- Verify Telegram bot tokens (`getMe`).
- Fetch recent updates (`getUpdates`).
- Keep history in SQLite DB.
- Restrict access with a secret key.
